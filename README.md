# GemLedger Working App

A Vite + React + Tailwind implementation of the GemLedger dashboard prototype.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown by Vite.

## Build

```bash
npm run build
```

## Deploy

Import this folder into Vercel, Netlify, StackBlitz, Replit, or GitHub Codespaces.
